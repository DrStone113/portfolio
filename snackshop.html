Features

🛒

Shopping Cart

Add, update quantities, remove items. Real-time cart total calculation.

💳

Checkout

Payment method selection, order confirmation, atomic SQL transaction.

📦

Order History

Track all past orders with status updates from admin.

🔐

Auth & Roles

SHA256 password hashing. Customer, Staff, Admin role-based access.

🏪

Admin Panel

Manage products, orders, promotions, customers from a dedicated panel.

⚡

Performance

HTTP gzip compression, static asset caching (7-day max-age), connection pooling.
