Planex

A personal task manager with a brain. Smart free-slot finder, Firebase push notifications, and a clean calendar view — all in PHP.

PHP 8 MySQL Firebase FCM HTML · CSS · JS
View Live Site ↗ View on GitHub

Features

✅

Task Management

Create, edit, delete tasks with priority levels (High/Medium/Low) and status tracking.

🔍

Smart Time-Slot Finder

Automatically finds free windows in your schedule based on existing tasks.

📆

Calendar View

Monthly calendar with task indicators and visual urgency highlighting for tasks due within 48h.

🔔

Push Notifications

Browser-based push via Firebase FCM. Reminders 30 minutes before task start/end.

🎨

Custom Categories

Color-coded categories auto-created on registration. Fully customizable.

🛡️

Admin Console

View all tasks, manage users — add, lock/unlock, delete. Role-based access.
