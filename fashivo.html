Fashivo API v1.0
🔐 auth
POST/api/v1/auth/signupRegister a new user
POST/api/v1/auth/loginAuthenticate and receive token
GET/api/v1/auth/meGet current user
👗 product
GET/api/v1/productList all products
GET/api/v1/product/:idGet product details
POST/api/v1/productCreate product (admin)
🛒 cart
GET/api/v1/cartGet current user's cart
POST/api/v1/cartAdd item to cart

Features

🛍️

Browse & Filter

Filter products by category with TanStack Vue Query for server state caching.

🔐

Auth & Profile

JWT auth, password update, forgot/reset password flow.

🛒

Cart Management

Add, update quantities, remove items. Pinia state management.

🏪

Admin Panel

Add, edit, manage products and categories. Image upload via Multer.

📋

Swagger Docs

Full OpenAPI spec. Interactive API documentation at /api-docs.

🛡️

Rate Limiting

express-rate-limit + Zod validation on all endpoints.
